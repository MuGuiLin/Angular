ng-class 为html元素动态的绑定一个或多个 CSS 类 

			ng-class 指令用于给 HTML 元素动态绑定一个或多个 CSS 类。
			ng-class 指令的值可以是字符串，对象，或一个数组。
			如果是字符串，多个类名使用空格分隔。
			如果是对象，需要使用 key-value 对，key 是一个布尔值，value 为你想要添加的类名。只有在 key 为 true 时类才会被添加。
			如果是数组，可以由字符串或对象组合组成，数组的元素可以是字符串或对象。
				  
			 为指定元素设置：ng-class="red"
			
			隔行变色：
			 为偶数行设置: ng-class-even="red"
			 为奇数行设置: ng-class-odd="green"
			 
			   ng-class="{真：'做什么' , 假：'做什么'}[条件表达式]"；
			 例：ng-class="{true:'pink', false:'red'}[m.sex == '女']"
			 
			点击事件(传参) 加 样式修改：
			ng-click="djFN($index)" 
			ng-class="{true:'black'}[$index == i]">
