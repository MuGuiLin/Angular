1.2、子组件通过@Output 和 EventEmitter 装饰器来触发 事件驱动 父组件的方法

(不推荐用，还是用上面1那种好点) 

        
        // 1、在子组件中引入 Output 和 EventEmitter ;
        import  Component, OnInit, Input, Output, EventEmitter } from '@angular/core';
        
        // 2、在子组件中实例化 EventEmitter
        @Output() public varName = new EventEmitter(); //声明EventEmitter实例，注：varName是自定义的

        // 3、在子组件中通过 EventEmitter 对象的 varName 实例广播数据
        this.varName.smit('子组件中的数据')

        // 4、在父组件中接收子组子事件
